About project

This project was developed as part of the Seminar iz načrtovanja in razvoja programske opreme v telekomunikacijah course.
The main motivation behind the project is to assist users in making informed decisions when buying a used car in Slovenia. All data used is publicly available from the Ministry of Infrastructure (due to new regulations, data from 2022):

https://podatki.gov.si/dataset/rezultati-tehnicnih-pregledov-motornih-vozil

The application allows users to compare annual technical inspection results across different car brands and production years. The concept is inspired by the website

www.autolog.si
